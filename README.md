# Trello Telegram Bot

## Configuration

Create a `config.py` file with your credentials:

```python
# Trello API credentials
API_KEY = "1111111111111"
TOKEN = "22222222222222222222222"
TODO_LIST_ID = "333333333333"
DOING_LIST_ID = "444444444444"
DONE_LIST_ID = "55555555555555555"
MY_MEMBER_ID = "666666666666666666"

# Telegram Bot credentials
TELEGRAM_BOT_TOKEN = "777777777777777777777777777"
CHAT_IDS = ["8888888"]

# Allowed group ID
ALLOWED_GROUP_ID = -9999999

Replace the placeholder values with your actual credentials.
